1) Démarrage rapide

  1. Lancez app.py (double-clic sur start.bat ou commande python app.py).
  2. Ouvrez l'URL locale affichée (généralement http://127.0.0.1:8000).
  3. Ajoutez quelques entreprises puis des prospects (ou importez vos données JSON/Excel).
Les données sont stockées localement dans une base SQLite (sur votre PC). Rien n'est envoyé sur Internet. Un snapshot automatique est créé chaque jour au premier lancement.

2) Dashboard

Votre tableau de bord quotidien pour voir en un coup d'œil l'état de votre activité :

  • KPI du jour : nombre de prospects, push envoyés, RDV planifiés, relances en retard, avec indicateurs de tendance (↑/↓) par rapport à la période précédente.
  • Graphique semaine : activité push et notes d'appel sur les derniers jours.
  • Relances en retard : liste directe des prospects dont le suivi est dépassé, avec accès rapide à la fiche.
  • Fil d'activité : flux chronologique de vos dernières actions (push, notes, changements de statut).
  • RDV à venir : prochains rendez-vous planifiés.
  • Pipeline : répartition visuelle de vos prospects par statut.

3) Prospects

La page principale de gestion de vos prospects (managers, responsables, décideurs chez vos clients potentiels).

  • Filtres avancés : recherche texte, entreprise, statut, pertinence (1–5 étoiles), relance (due / a une date / aucune), "avec téléphone", "avec email", "push envoyé".
  • Tri : cliquez sur les en-têtes de colonnes (Nom, Entreprise, Pertinence, Statut, Date de push…).
  • Sélection multiple : cochez plusieurs prospects pour appliquer un statut ou une pertinence en masse, exporter, ou supprimer.
  • Vue Tableau / Kanban : basculez entre une vue tableau classique et une vue Kanban qui organise les prospects par statut sous forme de colonnes glisser-déposer.
  • Vues sauvegardées : enregistrez vos combinaisons de filtres et tris favorites pour les retrouver en un clic. Créez autant de vues que nécessaire (ex: "Mes prospects chauds Lyon", "Relances de la semaine").
  • Vue Contacts : activez le mode contacts (?contacts=1 dans la sidebar) pour un affichage orienté annuaire avec téléphone et email en avant.
  • Export : export Excel (.xlsx) de la liste filtrée avec toutes les colonnes et données enrichies.

4) Fiche prospect détaillée

Cliquez sur un prospect pour ouvrir sa fiche complète avec plusieurs onglets :

  • Infos : nom, fonction, entreprise, téléphone, email, LinkedIn, pertinence (étoiles), statut, tags techniques, notes libres.
  • Photo : uploadez une photo du prospect pour le reconnaître facilement.
  • Historique : timeline chronologique de toutes les interactions — push envoyés, notes d'appel, enrichissements IA.
  • Relance : définissez la prochaine date de relance (nextFollowUp). Le prospect apparaîtra dans la page Focus.
  • Catégorie Push : associez une catégorie push au prospect pour que le bon template Outlook s'ouvre automatiquement lors de l'envoi.
  • Meilleurs candidats : l'app compare automatiquement les tags du prospect avec les compétences de vos candidats et affiche les meilleurs matchs (score de correspondance en %).
  • Onglet RDV (si statut "Rendez-vous" ou "Rencontré") : checklist de qualification RDV avec des thèmes de discussion pré-configurés, barre de progression, et cases à cocher pour préparer et suivre vos rendez-vous.
  • Actions rapides : bouton email (copie + ouverture template Outlook), vCard, scrapping IA, marquer comme fait.

5) Scrapping IA (enrichissement)

Enrichissez vos fiches prospect, entreprise et candidat grâce à l'intelligence artificielle. Le processus est en 2 clics :

  1. 1er clic sur "🤖 Scrapping IA" : un prompt optimisé et pré-rempli avec les données actuelles de la fiche est copié dans votre presse-papier. Collez-le dans votre IA favorite (Perplexity, ChatGPT, Comet, GenSpark…).
  2. 2e clic sur "📥 Coller retour IA" : une modale s'ouvre pour coller le résultat de l'IA. Le système analyse automatiquement le texte retourné, détecte les champs (fonction, téléphone, email, LinkedIn, tags, notes…) et les affiche en prévisualisation.

Vous pouvez accepter ou ignorer chaque champ individuellement avant d'appliquer. L'enrichissement est tracé dans la timeline de l'entité avec le nombre de champs mis à jour.

  • Prospect : enrichit la fonction, téléphone, email, LinkedIn, tags techniques, métier, pertinence, notes, et infos entreprise.
  • Entreprise : enrichit le secteur, effectif, téléphone standard, tags, notes d'activité, et peut détecter des managers clés à créer automatiquement comme nouveaux prospects.
  • Candidat : enrichit le rôle, compétences techniques, séniorité, localisation, LinkedIn, disponibilité, et TJM.
Le prompt est optimisé pour le secteur systèmes embarqués / électronique / ingénierie et utilise les tags de votre référentiel métiers.

6) Entreprises

La liste de vos entreprises clientes / cibles, avec vue résumée :

  • Groupe + Site : organisez par groupe (ex: "Renault") et site/localisation (ex: "Vénissieux").
  • Tags techniques : stockez la stack technique de l'entreprise (C/C++, FPGA, AUTOSAR…). Ces tags sont utilisés par le moteur de matching sourcing.
  • Notes : contexte libre, interlocuteurs, pistes commerciales.
  • Résumé rapide : nombre de prospects, dernier push, RDV planifiés.
  • Fiche popup : ouvrez la fiche détaillée depuis la liste, sans quitter l'écran entreprises.
  • Scrapping IA : même workflow en 2 clics que pour les prospects.

7) Fiche entreprise détaillée

La fiche détaillée s'ouvre en popup depuis la page /entreprises et offre une vue complète de l'entreprise :

  • Informations : groupe, site, téléphone, tags, secteurs, notes — tout modifiable en ligne.
  • Prospects liés : liste de tous les prospects rattachés à cette entreprise, avec accès direct à chaque fiche.
  • Opportunités : gérez vos opportunités commerciales par entreprise. Chaque opportunité a un titre, un stade (Identification → Qualification → Proposition → Négociation → Gagné / Perdu), un candidat associé, un montant, et des notes. Idéal pour suivre vos placements en cours.
  • Timeline : historique chronologique de toutes les interactions — push, notes de prospects, événements, enrichissements IA.
  • Événements : ajoutez des événements manuels à la timeline (rencontres, appels, infos).

8) Focus & relances

La page Focus est votre to-do de prospection. Elle centralise tous les prospects ayant une date de relance (nextFollowUp) définie :

  • Segments : en retard (rouge), aujourd'hui (orange), demain, cette semaine, plus tard.
  • Actions rapides : décaler la relance, marquer "fait" (supprime la date de relance), ouvrir la fiche prospect.
  • Compteurs : nombre de relances par segment pour prioriser votre journée.
Conseil : à chaque interaction avec un prospect, définissez systématiquement la prochaine relance. Votre Focus devient votre agenda de prospection.

9) Calendrier

Vue calendrier mensuelle de vos RDV planifiés et relances :

  • Les RDV (prospects avec statut "Rendez-vous" et date RDV) sont affichés sur le calendrier.
  • Les relances planifiées (nextFollowUp) sont également visibles.
  • Cliquez sur un événement pour accéder directement à la fiche prospect.
  • Navigation mois par mois.

10) Sourcing candidats

Votre base candidats et outil de productivité pour le matching :

  • Pipeline candidats : liste complète avec filtres par statut (À sourcer, Contacté, Qualifié, En mission, Archivé), recherche, et tri.
  • Ajout rapide : créez un candidat avec nom, rôle, compétences, localisation, LinkedIn, source.
  • Import LinkedIn CSV : importez directement un export de contacts LinkedIn — le système mappe automatiquement les colonnes (prénom, nom, poste, URL…).
  • Export CSV : exportez votre base candidats.

Outil Productivité / Matching :

  • Sélection d'entreprises cibles : choisissez une ou plusieurs entreprises pour lesquelles vous cherchez des candidats.
  • Mots-clés automatiques : le système extrait automatiquement les tags techniques de l'entreprise pour générer des mots-clés de recherche. Vous pouvez ajouter/retirer des mots-clés manuellement.
  • Requête LinkedIn : une requête booléenne LinkedIn est générée automatiquement, prête à copier-coller dans LinkedIn Recruiter / Sales Navigator.
  • Matching candidats : score de correspondance entre chaque candidat et les mots-clés sélectionnés.
  • Candidat du jour ⭐ : le meilleur match selon les filtres actuels est mis en avant, avec ouverture directe de son profil LinkedIn.
  • Message d'approche : génération d'un message d'approche personnalisé, prêt à coller.
  • Sauvegarde tags : enregistrez les mots-clés productivité directement dans les tags de l'entreprise pour enrichir les futures recherches.

11) Fiche candidat détaillée

Chaque candidat a sa fiche complète accessible depuis la page sourcing :

  • Informations : nom, rôle/poste, localisation, séniorité, technologie principale, LinkedIn, téléphone, email, source, TJM, disponibilité.
  • Compétences (skills) : tags techniques avec saisie intelligente et autocomplete depuis le référentiel métiers.
  • Statuts : À sourcer → Contacté → Qualifié → En mission → Archivé (avec archivage/désarchivage).
  • Entreprises liées : associez un candidat à une ou plusieurs entreprises cibles avec recherche et sélection multiple.
  • Notes : zone de texte libre pour le suivi.
  • Auto-save : les modifications sont sauvegardées automatiquement après quelques secondes d'inactivité.
  • Scrapping IA : même workflow en 2 clics (copie prompt → colle résultat IA) pour enrichir automatiquement le profil du candidat.

12) Push & envoi d'emails

Le push désigne l'envoi d'un email/CV à un prospect. Le workflow est optimisé :

  1. Depuis la fiche prospect, cliquez sur 📧 Email.
  2. L'adresse email est copiée automatiquement dans le presse-papier.
  3. Si une catégorie push est associée au prospect, le template Outlook (.msg) correspondant est ouvert automatiquement. Vous n'avez qu'à coller l'email dans le champ "À:".
  4. Le push est loggé automatiquement dans l'historique avec la date, le canal et le template utilisé.

Page Push (historique) :

  • Historique complet de tous les push envoyés, filtrables par recherche et canal.
  • Annuler le dernier push : en cas d'erreur, annulez le dernier envoi.
  • Export Excel : exportez l'historique des push au format .xlsx.
  • Suppression unitaire si besoin.

13) Catégories Push

Organisez vos templates d'emails push par compétence / métier :

  • Catégories : chaque catégorie correspond à un dossier dans pushs/ (ex: "C_embarqué", "FPGA", "Electronique"). Le système scanne automatiquement les dossiers existants.
  • Fichiers .msg : chaque catégorie contient des fichiers templates Outlook (.msg, .eml, .oft). Le premier fichier est utilisé comme template par défaut lors de l'envoi.
  • Mots-clés : associez des mots-clés à chaque catégorie pour le matching automatique.
  • Matching candidats : chaque catégorie affiche les Top 3 candidats dont les compétences correspondent le mieux aux mots-clés de la catégorie (score de correspondance).
  • Scan dossiers : le bouton "Scanner" détecte automatiquement les nouveaux dossiers dans pushs/ et crée les catégories correspondantes.
  • Éditeur de templates : créez/modifiez des templates email avec éditeur rich text (gras, italique, liens, variables).

14) Recherche globale

Ouvrez la recherche rapide avec Ctrl+K (ou cliquez sur le badge 🔍 dans la sidebar) :

  • Recherche instantanée dans : prospects, entreprises, candidats, push logs.
  • Résultats avec mise en surbrillance du terme recherché.
  • Navigation au clavier (↑↓ + Entrée) pour ouvrir directement la fiche.
  • Accès direct : chaque résultat renvoie vers la bonne page (fiche prospect, fiche entreprise, fiche candidat, page push).

15) Statistiques

Analyse visuelle de votre activité de prospection :

  • KPIs : push envoyés, notes d'appel, relances en retard, taux de conversion, sur période configurable (semaine, mois, trimestre…).
  • Graphique statuts : répartition de vos prospects par statut (doughnut chart).
  • Graphique pertinence : répartition par niveau de pertinence.
  • Activité push : courbe d'évolution des push envoyés par semaine/mois.
  • Top entreprises : classement des entreprises les plus actives (basé sur nombre de push, RDV, et interactions).
  • Filtrage temporel : choisissez la plage (semaine, mois, trimestre, année) et naviguez entre les périodes.

16) Rapport hebdomadaire

Générez un rapport de synthèse pour chaque semaine :

  • KPIs hebdo : push envoyés, notes d'appel, nouveaux prospects, RDV, relances traitées.
  • Détail par entreprise : activité détaillée par client avec tags, push, et interactions de la semaine.
  • Barre de statuts : répartition visuelle de votre pipeline.
  • Export Markdown : générez le rapport au format Markdown, prêt à copier-coller dans OneNote, Slack, Teams, ou un email.
  • Impression : mode impression optimisé (CSS print).
  • Navigation semaine : sélecteur de semaine pour consulter les rapports passés.

17) KPI & export Excel de validation

Outil de checklist rapide pour générer un fichier Excel de validation :

  • Sélectionnez une date et un titre (ex: "Semaine 06").
  • Cliquez sur "Générer Excel" pour obtenir un .xlsx formaté avec vos KPIs du jour, prêt à envoyer à votre manager ou à archiver.

18) Référentiel métiers

Le référentiel contient 251 tags techniques organisés en 15 catégories métier (Électronique, Logiciel embarqué, Informatique industrielle, Mécanique, Systèmes, Automatisme, Robotique, etc.) :

  • Arbre interactif : parcourez les catégories et sous-catégories sous forme d'arborescence dépliable.
  • Recherche : filtrez rapidement pour trouver un tag ou un métier.
  • Statistiques : nombre de tags par catégorie, couverture dans votre base.
  • Tags personnalisés : ajoutez vos propres tags métier en complément du référentiel standard.
  • Utilisation : ces tags sont utilisés partout dans l'app — tags prospects, tags entreprises, compétences candidats, mots-clés de catégories push, et prompts de scrapping IA.

19) Doublons

  • Détection : le système identifie les doublons potentiels par nom, email, ou téléphone.
  • Fusion : pour chaque paire, choisissez la fiche à garder et celle à fusionner. Les données sont consolidées (tags, notes, callNotes, push logs) dans la fiche conservée.
  • Prévisualisation : comparez les deux fiches côte à côte avant de fusionner.

20) Snapshots & restauration

  • Snapshot manuel : créez une sauvegarde instantanée à tout moment.
  • Snapshots automatiques : un snapshot est créé automatiquement au premier lancement de chaque jour.
  • Restauration : restaurez n'importe quelle version. Une sauvegarde "before_restore" est créée automatiquement avant chaque restauration pour sécuriser l'opération.
  • Historique : consultez la liste complète des snapshots avec date et label.
Conseil : faites un snapshot avant une grosse importation ou un nettoyage de données.

21) Paramètres & exports

La page Paramètres regroupe tous les outils de gestion de données :

  • Export JSON : sauvegardez l'intégralité de votre base au format JSON (compatible ré-import).
  • Import JSON : restaurez une base depuis un export JSON précédent.
  • Export Excel : export complet au format .xlsx avec toutes les colonnes.
  • Import tags CSV : importez des tags depuis un fichier CSV pour enrichir vos prospects/entreprises en masse.
  • Export vCard (.vcf) : exportez tous les prospects ayant un numéro de téléphone au format vCard, compatible iPhone/iCloud et tout gestionnaire de contacts.
  • Réinitialiser : remise à zéro de la base (avec confirmation).
  • Accès rapide : liens vers Snapshots, Métiers, Doublons, Aide.

22) Système de mises à jour

Mettez à jour l'application directement depuis la page Paramètres :

  • Glisser-déposer : déposez vos fichiers de mise à jour (.py, .html, .js, .css…) dans la zone prévue. Le système détecte automatiquement l'emplacement de chaque fichier dans le projet.
  • Zone de staging : vérifiez les fichiers détectés et leur destination avant d'appliquer.
  • Backup automatique : une sauvegarde complète du projet est créée automatiquement avant chaque mise à jour.
  • Application : un clic sur "Appliquer" remplace les fichiers. Relancez ensuite l'application pour que les changements prennent effet.
  • Rollback : si une mise à jour pose problème, restaurez une version précédente en un clic depuis la liste des sauvegardes.

23) Raccourcis & interface

  • Ctrl+K : recherche globale rapide.
  • Thème clair/sombre : basculez via le bouton 🌙 dans la sidebar. Le choix est mémorisé.
  • Sidebar rétractable : cliquez sur le chevron pour réduire la sidebar et gagner de l'espace.
  • Mobile responsive : menu hamburger sur mobile, interface adaptée aux petits écrans.
  • Notifications toast : les actions (sauvegarde, export, erreur…) sont confirmées par des notifications visuelles en bas à droite.
  • Tag autocomplete : lors de la saisie de tags, des suggestions issues du référentiel métiers apparaissent automatiquement.
  • Bouton recherche flottant : sur mobile, un bouton flottant 🔍 permet d'ouvrir la recherche globale.

24) Bonnes pratiques

  • Renseignez nextFollowUp à chaque interaction → votre page Focus devient votre to-do quotidien.
  • Centralisez la stack technique dans les tags entreprise pour booster le matching sourcing et la génération de requêtes LinkedIn.
  • Utilisez le Scrapping IA systématiquement sur les nouvelles fiches pour les enrichir rapidement.
  • Associez une catégorie push à chaque prospect pour automatiser l'ouverture du bon template Outlook.
  • Faites un Snapshot avant toute importation ou nettoyage massif.
  • Utilisez les vues sauvegardées pour vos scénarios de filtrage récurrents.
  • Consultez le Dashboard chaque matin pour prioriser votre journée.
  • Utilisez la Recherche Ctrl+K plutôt que naviguer dans les listes.
  • Exportez un Rapport hebdo en fin de semaine pour votre suivi d'activité.
  • Tenez à jour les Opportunités sur les fiches entreprise pour suivre votre pipeline commercial.
