🔔 Notifications navigateur

Rappel quotidien (relances en retard / à faire aujourd'hui) via une notification du navigateur, une fois par jour à l'heure choisie. L'onglet ou l'app doit avoir été ouverte récemment.

🏷️ Import de tags depuis CSV

Importe des compétences/tags en masse depuis un fichier CSV ou Excel. Colonnes acceptées : nom (ou name), entreprise (ou company), tags (ou tag, compétences).

Séparateur détecté automatiquement (point-virgule, virgule ou tabulation). Si les accents sont mal affichés, enregistrez le fichier en UTF-8.

Matching approximatif (ignore casse/accents) par nom + entreprise. Les tags sont fusionnés avec les existants.

📱 Export contacts iPhone

Exporte tous les prospects qui ont un numéro de téléphone au format vCard (.vcf), compatible iPhone (Contacts) et import iCloud.

Inclut : Nom, Entreprise (ORG), Téléphone (CELL), Email (si présent) et Fonction (si présent).

📂 Dossier candidats

Configurez le chemin vers les dossiers candidats sur votre disque (CV, documents…).

Le dossier racine contenant tous les sous-dossiers candidats.
Variables : {NOM} (majuscules), {Nom} (capitalisé), {nom} (minuscules), {Prenom} / {PRENOM} / {prenom}.
Ex: {NOM} {Prenom} → GORSE Dorian · {Nom}_{Prenom} → Gorse_Dorian

🔗 Teams / Planner

Préfixe utilisé dans les copies pour Teams Planner (ex: ABI pour Antoine Binet). Ce trigramme est ajouté devant les noms d'entreprise lors des pushes.

2 à 5 caractères (lettres/chiffres). Ex : ABI, PAD, TZA, MLA

📡 Webhook automatique

Envoie automatiquement une notification dans Teams quand vous faites un push, prenez un RDV, ajoutez un CR ou créez une tâche.
Comment : Dans Teams → onglet « Workflows » → créer « Post to a channel when a webhook request is received » → copier l'URL ici.

📋 Template InMail LinkedIn

Ce texte sera copié automatiquement lors d'un push LinkedIn. Variables disponibles : {nom}, {prenom}, {entreprise}, {fonction}, {civilite}
